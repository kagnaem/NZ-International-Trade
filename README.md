# New Zealand International Trade Data Analysis

## Project Overview
This repository presents an exploratory analysis of New Zealand international trade data using Python. The project focuses on understanding trade patterns, numeric relationships, and year-based variation in imports, exports, and CPI-related data.

## Background
This project began as a school assignment and was originally developed using MS Access, Excel, Power BI, and DAX. The GitHub version extends that work into a Python-based workflow that can be reviewed directly through the notebook and supporting project files.

## Data Source
The dataset used in this project is a combined CSV built from:
- Stats NZ
- OECD
- IMF

The final analysis dataset is available in this repository as:
- [NZInternationalTradewithCPI.csv](./NZInternationalTradewithCPI.csv)

## Main Project Files
- [NZ IntTrade_Analysis.ipynb](./NZ%20IntTrade_Analysis.ipynb): the main exploratory analysis notebook
- [NZInternationalTradewithCPI.csv](./NZInternationalTradewithCPI.csv): cleaned dataset used in the notebook
- [Em-NZInternationalTradewithCPI_Data Analysis.xlsx](./Em-NZInternationalTradewithCPI_Data%20Analysis.xlsx): workbook version of the project data and analysis
- [Em 2024 New Zealand International Trade -Part1-3.pdf](./Em%202024%20New%20Zealand%20International%20Trade%20-Part1-3.pdf): full project paper

## Notebook Coverage
The notebook has been structured as a complete project document for GitHub readers. It includes:
- project introduction and workflow
- data loading and cleaning
- descriptive statistics
- continent-level distribution checks
- correlation analysis
- trade distribution charts
- time-based trade trend visuals
- yearly time-series analysis for imports, exports, and CPI
- year-by-year boxplot comparisons
- result interpretation and insights after each major section

## Time-Series Analysis
Using the project paper as a broader reference point, the notebook now includes a dedicated time-series section to show how average imports, exports, and CPI move from year to year.

This addition helps readers:
- connect the exploratory analysis to time-based trade behaviour
- see how imports and exports change across the observed period
- understand CPI movement alongside trade values
- follow the project more easily as a complete analytical narrative on GitHub
